Documentación Técnica de TactiLink
===================================

Bienvenido a la documentación técnica de **TactiLink**, un sistema de transcripción de texto a Braille y generación de señalética Braille en formato SVG.

Descripción General
-------------------

TactiLink proporciona una API REST que permite:

- **Transcripción de texto a Braille**: Convierte texto en español a código Braille numérico
- **Generación de señalética**: Crea representaciones visuales en SVG de señalética Braille con puntos táctiles
- **Soporte multilingüe**: Maneja caracteres especiales del español (acentos, ñ, etc.)
- **Contexto numérico**: Detecta y maneja automáticamente secuencias de números

.. note::
   
   La API está disponible en ``http://localhost:5000`` cuando el servidor está ejecutándose.
   Todos los endpoints aceptan JSON y responden en JSON.

Características Principales
---------------------------

✓ API REST basada en Flask
✓ Soporte para letras (a-z), números (0-9) y signos de puntuación
✓ Generación de SVG escalable para señalética Braille
✓ Endpoints de salud para monitoreo
✓ CORS habilitado para integración con frontend

Inicio Rápido
-------------

Para usar la API:

1. **Transcribir texto a Braille**:

   Convierte palabras en español a códigos Braille numéricos.

   .. code-block:: bash

      curl -X POST http://localhost:5000/api/transcribe \
        -H "Content-Type: application/json" \
        -d '{"text": "Hola"}'

   Respuesta esperada: Códigos Braille numéricos

2. **Generar señalética SVG**:

   Crea un archivo gráfico que puede imprimirse para lectura táctil.

   .. code-block:: bash

      curl -X POST http://localhost:5000/api/generar_senaletica \
        -H "Content-Type: application/json" \
        -d '{"text": "Hola"}'

   Respuesta esperada: SVG con puntos táctiles

3. **Verificar salud del servicio**:

   Confirma que el servidor está activo y funcionando.

   .. code-block:: bash

      curl http://localhost:5000/health

   Respuesta esperada: {"status": "ok"}

Requisitos
----------

Para ejecutar TactiLink necesitas:

- **Python 3.8+**: Lenguaje de programación para la API
- **Flask**: Framework web que proporciona la API
- **Flask-CORS**: Permite que el frontend acceda a la API
- **svgwrite**: Librería para crear archivos gráficos SVG

Todos estos se instalan automáticamente en el siguiente paso.

Instalación
-----------

Sigue estos pasos para que TactiLink funcione en tu computadora:

.. code-block:: bash

   # Paso 1: Instala todas las librerías necesarias
   pip install -r requirements.txt
   
   # Paso 2: Inicia el servidor
   python app.py

Listo. El servidor estará activo en http://localhost:5000

.. warning::

   Asegúrate de estar en el directorio correcto antes de ejecutar estos comandos.
   Si encuentras errores, revisa que Python 3.8+ esté instalado.
