# Acceso a la Documentación Técnica

## Ubicación

La documentación técnica de TactiLink está disponible en formato HTML interactivo. Para accederla, dirígete al siguiente archivo:

```
DocumentacionTecnica/html/index.html
```

## Cómo Acceder

1. **Navega a la carpeta** `DocumentacionTecnica/html/`
2. **Abre el archivo** `index.html` en tu navegador web
3. Verás la documentación completa de la API con ejemplos y referencias

## Contenido de la Documentación

La documentación incluye:

- **Referencia de API**: Descripción completa de todos los endpoints
- **Funciones principales**: Detalles técnicos de cada función
- **Ejemplos de uso**: Casos prácticos de implementación
- **Parámetros y retorno**: Información detallada de inputs/outputs
